# NoodleFit

## What it is

A Runkeeper clone.

## Installation instructions

Clone the repository

> git clone git://github.com/nooodle/noodlefit.git

Install node by using brew or through the website http://nodejs.org/#download

> cd noodlefit

> cp local.json-dist local.json

> npm install

Run the site

> node app.js

## Tests

> make test
